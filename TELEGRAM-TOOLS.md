# Telegram — отправка и приём медиа

## Отправка медиа клиенту

Выведи маркер в тексте ответа — бот автоматически отправит файл в Telegram.

### Синтаксис
```
[ТИП: путь_или_URL подпись]
```

Подпись опциональна (через пробел). Маркер удаляется из текста.

### Типы

| Маркер | Что отправляет | Лимит |
|--------|---------------|-------|
| `[ФОТО: /tmp/img.jpg]` | Фото (сжатое) | 10MB |
| `[ФАЙЛ: /tmp/report.pdf]` | Документ (без сжатия) | 50MB |
| `[СТИКЕР: /tmp/sticker.webp]` | Стикер | 512KB |
| `[ВИДЕО: /tmp/video.mp4]` | Видео | 50MB |
| `[АУДИО: /tmp/track.mp3]` | Аудио (с плеером) | 50MB |
| `[ГОЛОС: /tmp/voice.ogg]` | Голосовое сообщение | 50MB |
| `[GIF: /tmp/animation.gif]` | GIF-анимация | 50MB |

Английские варианты: PHOTO, FILE, STICKER, VIDEO, AUDIO, VOICE, ANIMATION.

### Примеры
```
[ФОТО: /tmp/chart.png Статистика за неделю]
[ФАЙЛ: /tmp/report.pdf]
[ГОЛОС: /tmp/summary.ogg Краткое резюме]
[ВИДЕО: /tmp/demo.mp4 Демонстрация]
```

URL тоже работают:
```
[ФОТО: https://example.com/image.jpg]
```

## Приём медиа от клиента

| Тип | Что получаешь |
|-----|--------------|
| Фото | Мультимодальный контент (base64) — видишь изображение |
| Фото как документ | То же — мультимодальный контент |
| Видео | Транскрипт + ключевые кадры |
| Голосовое | Транскрипт (Deepgram) |
| Стикер | Изображение (мультимодальный) |
| Документ | Сохранён в `~/workspace/.media/`. Текст < 20KB — инлайнится |

### Работа с документами

Файлы сохраняются в `~/workspace/.media/doc_{timestamp}_{filename}`.

- **Текстовые < 20KB** (.txt, .md, .json, .csv, .py и др.) — содержимое инлайнится в промпт + путь
- **Текстовые > 20KB** — только путь. Открой через Read tool
- **PDF** — только путь. Читай через Read tool (до 20 страниц за вызов)
- **XLSX/DOCX** — только путь. Используй Bash для парсинга (node скрипты, npm пакеты)
- **Любые другие** — сохранены на диск, путь передан. Решай сам как обработать

## Ограничения Telegram Bot API

- Скачивание файлов: максимум 20MB
- Отправка фото: максимум 10MB
- Отправка остальных: максимум 50MB
- Текст сообщения: максимум 4096 символов (используй 4000 для безопасности)
- Подпись к медиа: максимум 1024 символа
