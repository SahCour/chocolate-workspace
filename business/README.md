# 🏪 Офлайн-бизнес: Структура и стратегия

## Цели

1. **Раскрутка Instagram/Facebook** — существующие аккаунты, живой контент, без ИИ-тона
2. **B2B-промо** — отели, рестораны, создание промо-материалов
3. **Боты для бизнеса** — Square-интеграция + учёт + коммуникация
4. **Какао-новостник** — бот-агрегатор трендов и «тихий» маркетолог
5. **ИИ-маркетолог** — исследование темы трафика для офлайна
6. **Перенос данных** — с диска и NotebookLM

## Структура

```
business/
├── README.md
├── social-media/    ← SMM (Instagram + Facebook)
├── b2b-promo/       ← отели, рестораны
├── bots/            ← Square-бот + какао-новостник
├── research/        ← исследования
├── data/            ← данные бизнеса (продукты, финансы, контакты)
├── data-transfer/   ← план переноса
└── ideas/           ← сырые идеи
```
