# 💾 Перенос данных — план

## Источники
1. **Локальный физический диск** — себестоимость, финданные, контакты
2. **Google NotebookLM** — технологические карты, рецептуры
3. **Локальные файлы** — фото, видео, промо

## Вопросы
- ❓ Как передать данные из NotebookLM? (экспорт → текст/CSV → файлы)
- ❓ Как передать данные с диска? (загрузка на сервер? Google Drive? Notion?)
- ❓ Сколько места нужно? ~100-200MB текст + фото/видео до 5-10GB
- ❓ Структура: по продуктам? по датам? по процессам?

## Предлагаемая структура
```
data/
├── products/         ← по продукту: себестоимость, рецепт, поставщики, фото
├── financials/       ← продажи, затраты, маржинальность
├── contacts/         ← отели, рестораны, поставщики, партнёры
└── operations/       ← инвентаризация, Square export, персонал
```

## Первый шаг (что загрузить в первую очередь)
1. Технологические карты (NotebookLM → текст/PDF)
2. Себестоимость продуктов (диск → CSV)
3. Контакты отелей/ресторанов
4. Топ-20 фото/видео
