# Какао-новостник

## Концепция
Бот-агрегатор о мире какао: от ферм до рецептов. Связывает поставщиков. «Тихий» маркетолог нашего бизнеса.

## Аудитория
- Поставщики какао
- Производители шоколада
- Шеф-кондитеры и рестораторы
- Энтузиасты

## Формат
- Telegram-бот (приоритет) + веб-версия
- Язык: английский (глобально)
- Daily digest / weekly newsletter

## Рубрики
1. 🌱 Фермы — урожай, регионы, устойчивость
2. 📈 Тренды — цены, спрос, новинки
3. 🍫 Рецепты — подача, сочетания, техники
4. 🔗 События — конференции, воркшопы
5. 🤝 Партнёрства — возможности для поставщиков

## Маркетинговая ценность
- Авторитетный источник → доверие
- Ненавязчивое продвижение нас
- Контакты с поставщиками
- Партнёрский контент (monetisation)
