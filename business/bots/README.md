# Боты для бизнеса

- Учёт остатков в реальном времени
- Канал коммуникации для персонала
- Сводка магазина на острове
- **Статус:** есть начальные шаги, код передадут

## 2. Какао-новостник

- Связь поставщиков
- Тренды: от ферм до рецептов
- «Тихий» маркетолог
- **Статус:** идея, концепция
