# Сценарий использования моделей

## Базовая модель: DeepSeek V4 Flash

**90% задач** — Flash. Дешёво, быстро, уверенно.

### Архитектура
- MoE, 284B параметров / 13B активных
- Контекст: 1M токенов
- text → text (изображения НЕ читает)
- Prompt: $0.09/M · Completion: $0.18/M

### Сильные стороны
- Скорость, дешевизна, инструменты (tool use), структурированные ответы
- Следование инструкциям, базовое reasoning
- Выгоден cache read ($0.02/M)

### Когда Flash НЕДОСТАТОЧЕН
1. **Глубина >5 шагов** — многоэтапный анализ, финансовая модель, юр. разбор
2. **Сложная математика/логика** — взаимозависимые переменные, длинные расчёты
3. **Креативная глубина** — стратегия, narrative, нешаблонные концепты
4. **Критичная точность** — бухучёт, договоры, тех. спецификации
5. **Зашумлённый контекст** — много противоречий, нужно разрешать

### Признаки «тупления» Flash
- Ответ короче/беднее запрошенного
- Пропущены пункты многосоставного запроса
- Противоречия внутри ответа
- «Правильно по форме — пусто по содержанию»
- Повтор вместо развития

## Сценарий переключения

### Правило
**Flash → 90% задач. Pro/альтернативы → 10%.**

### Критерии

| Ситуация | Модель |
|---|---|
| Сложный analysis, >5 шагов | DeepSeek V4 Pro или Qwen 3.7 Plus |
| Финансы/налоги/договоры | DeepSeek V4 Pro или Qwen 3.7 Plus |
| Стратегия, документ >5 стр | DeepSeek V4 Pro или Qwen 3.7 Plus |
| Flash тупит (повтор) | DeepSeek V4 Pro |
| **Фото/скриншот/график** | **Gemini 3.1 Flash Lite** (основная) |
| PDF как изображение | Gemini 3.1 Flash Lite |
| Bulk-задачи, малый контекст | GLM 4.7 Flash |
| Deep reasoning, 1M контекст | GLM 5.2 |

### Как работает на практике
- Zen сам оценивает задачу
- Если задача complex — пишет: «Тут нужен Pro для точности, переключаю?»
- Если нужно прочитать изображение — переключает на Gemini 3.1 Flash Lite
- Если Flash уже тупанул — повтор на Pro

### Техническая реализация
- `/model openrouter/...` — ручное переключение
- В перспективе: авто-роутинг через OmniRoute

### Экономика
При дне: 100K prompt + 10K completion на Flash (~$0.011/день).
Pro для 10% задач: +~$0.005–0.015.
**Итого: $0.5–1.5/мес** против $4.50/мес если сидеть на Pro всё время.
